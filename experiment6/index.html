文本域---text

  将<input>标记中的 type 属性值设置为 text，就可以在表单中插入文本框。在此文本框中可以输入任何类型的数据，但输入的数据将以单行显示。其中，name 属性用于定义文本框的名称，maxlength 和 size 属性用于指定文本框的宽度和允许用户输入最大的字符数，更多情况下，采用 CSS 设置。value 指定文本框的默认值。

  pattern 属性规定用于检查<input>元素值的正则表达式。pattern 属性适用于以下输入类型：text、search、url、tel、email、and password。

按钮---button
